关于 HeaderPilot 浏览器扩展如何处理您的数据

How the HeaderPilot browser extension handles your data

生效日期：Effective date: 2026-07-13

01 总览 Overview

HeaderPilot（以下简称"本扩展"）是一款用于编辑 HTTP 请求头与响应头的浏览器扩展程序。我们非常重视您的隐私，本政策说明本扩展在您使用过程中如何处理数据——简单来说：我们不收集，也不上传任何数据。

HeaderPilot ("the extension") is a browser extension for editing HTTP request and response headers. We take privacy seriously, and this policy explains how data is handled while you use it — in short: we don't collect or upload any data.

02 我们不收集的内容 What we don't collect

  • 不收集您的浏览历史、访问的网址或页面内容
  • No browsing history, visited URLs, or page content
  • 不收集个人身份信息（姓名、邮箱、账号等）
  • No personally identifiable information (name, email, account, etc.)
  • 不读取、拦截或上传请求/响应的正文内容
  • No reading, intercepting, or uploading of request/response bodies
  • 不使用任何第三方分析、统计或广告 SDK
  • No third-party analytics, telemetry, or advertising SDKs

03 本地存储的数据 Data stored locally

您在弹出窗口中创建的自定义 Header 规则（如 Header 名称、值、匹配的网址模式、启用状态）会通过浏览器提供的 chrome.storage.local 接口，仅保存在您本机浏览器的本地存储中。

Custom header rules you create in the popup (header name, value, URL match pattern, enabled state) are stored only on your local machine via the browser's chrome.storage.local API.

这些数据：

This data:

  • 不会同步到任何 Anthropic、Google 或第三方服务器（除非您主动开启浏览器自带的 Chrome 账号同步功能）
  • Is never sent to any Anthropic, Google, or third-party server (unless you enable Chrome's own account sync feature)
  • 仅在您卸载扩展或手动清除浏览器数据时被删除
  • Is only removed when you uninstall the extension or manually clear browser data
  • 可通过扩展内置的"导出"功能，由您自行导出为本地 JSON 文件
  • Can be exported to a local JSON file at any time via the built-in "Export" feature

05 数据的删除 Deleting your data

由于所有数据都只存储在您本机，您可以随时通过以下方式清除：卸载扩展；或在扩展弹出窗口中逐条删除规则；或通过浏览器的"扩展程序数据"管理界面清除本地存储。

Since all data lives only on your device, you can remove it at any time by: uninstalling the extension, deleting rules one by one in the popup, or clearing local storage via your browser's extension data management screen.

06 政策变更 Changes to this policy

若本政策内容发生变更，我们会更新本页面顶部的生效日期。建议您在更新扩展版本后回到本页面查看最新内容。

If this policy changes, we will update the effective date at the top of this page. Please revisit this page after updating the extension to review the latest version.

If you have any questions about this privacy policy, please reach out via the developer contact listed on the Chrome Web Store listing.
